Alex Nelson | Madison Area Crime Visualization

Madison Area Crime Visualization

Type: Interactive Web Map

Library used: Leaflet

Program language used: HTML, JavaScript, CSS, jQuery

Description: This is my second ever attempt at creating an interactive web map using Leaflet. This is a project I did my final semester at UW-Madison for an Independent Study credit under Tanya Buckingham. Since this my second attempt at using Leaflet, I wanted to expand my knowledge and integrate features that I learned using D3 (ex. dropdown menus, display data using a choropleth) into a map using Leaflet. My goal for this web map, is to allow for users to visualize crimes through various attributes, whether it be time of day, location of incident, type of incident, etc. I'm hoping people will be able to use the data in order to become more aware and be safer when walking the streets at night or during the day.
